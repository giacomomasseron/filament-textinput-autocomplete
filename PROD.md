---
name: filament_textinput_autocomplete
description: "Plugin for FilamentPHP version > 5 that implements a TextInput with autocomplete feature"
version: 1.0
---

# Specifications

- Override TextInput filament field  
- Get autocomplete entries both statically and from API  
- It can use a view, a string or HTML to format any entry in autocomplete list  
- Check examples in examples/ folder to see the final result and how to howto use Livewire.js
